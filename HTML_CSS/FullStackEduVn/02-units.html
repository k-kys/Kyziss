CSS Units

Absolute units

  • px: pixels (1px=1/96th of 1in)
  • pt : points (1pt=1/72 of 1in)
  • cm : centimeters
  • mm : millimeters
  • in : inches (1in = 96px = 2.54cm)
  • pc : picas (1pc=12pt)

Relative units (cần nơi để phụ thuộc)

  • % - phụ thuộc vào thẻ cha (thẻ chứa nó)
  • rem - phụ thuộc vào thẻ html - dùng nhiều hơn em
  • em - phụ thuộc thẻ (cha) gần nhất chứa nó có cùng thuộc tính
  • vw: viewport width - phụ thuộc vào chiều rộng khung nhìn (toàn bộ chiều ngang trình duyệt)
  • vh: viewport height - tương tự vw nhưng phụ thuộc vào chiều dài khung nhìn
  • vmin
  • vmax
  • ex
  • ch
