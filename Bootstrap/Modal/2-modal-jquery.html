Bạn có thể sử dụng JQuery để tương tác với Modal, chẳng hạn ẩn hoặc hiện Modal.

$('#myModal').modal('toggle');
$('#myModal').modal('show');
$('#myModal').modal('hide');

Hoặc đầy đủ hơn:

        $('#myModal').modal(options); 
        // Example: 
        var options = { 
            'backdrop' :'static', 
            'keyboard' : true, 
            'show' : true, 
            'focus' : false }
